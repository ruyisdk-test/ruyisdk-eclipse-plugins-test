# 创建虚拟环境并应用到项目

## 操作步骤

1. 打开或创建一个 RISC-V 架构的项目
2. 点击导航栏 RuyiSDK 中 的 Venv 视图
3. 创建新的虚拟环境
4. 点击 Apply to Project 应用

## 预期结果

可以正常进行创建虚拟环境并应用到项目

## 实际结果

ruyi 版本： 0.43.0  
可以正常进行创建虚拟环境并应用到项目
